# Rails-Angular-Todo-List

Lista de tarefas usando rails como backend e angular como front

## Instruções pra Backend:

Instale os módulos:

```bundle install```

Faça as migrações do DB:

```rails db:migrate```

Depois rode o servidor:

```rails s```


## Instruções pra Frontend:

Instale os módulos:

```npm i```

Depois rode o servidor:

```ng serve```
